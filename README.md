Este é o repositório do meu portfólio pessoal. Um ambiente desenvolvido para centralizar meus projetos, experiências profissionais e stack tecnológica, focado em **Qualidade de Software (QA)** e **Desenvolvimento Front-end**.

## 💻 Sobre o Projeto

O projeto foi construído utilizando as tecnologias mais modernas do ecossistema React, priorizando performance, acessibilidade e uma interface intuitiva com temática *dark mode*.

### Principais Funcionalidades:
- **Design Responsivo:** Adaptado para dispositivos mobile, tablets e desktop.
- **Seção de Experiência:** Timeline interativa com acordeão para detalhes de atividades.
- **Filtro de Projetos:** Sistema dinâmico para filtrar projetos por categoria.
- **Animações Fluidas:** Utilização de Framer Motion para transições de estados e scroll.

## 🛠️ Tecnologias Utilizadas

- **Framework:** [Next.js 14/15](https://nextjs.org/)
- **Linguagem:** [TypeScript](https://www.typescriptlang.org/)
- **Estilização:** [Tailwind CSS](https://tailwindcss.com/)
- **Ícones:** [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)
- **Animações:** [Framer Motion](https://www.framer.com/motion/)
